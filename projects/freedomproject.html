Freedom Project using Earsketch

For this project I made a tiny snippet using Earsketch which is a Music making tool that utilizes JavaScript to work and produce music. The song that I created fit into a game soundtrack and you'll be able to play it with the link down below :)

Challenges and Takeaways

Some challenges that I had for this project was one, I spent too much time looking for sounds rather than focusing on the actual coding. This is due to the fact that it was a tool that had limits so choosing the right sounds for me was very important.

One takeaway for me personally is to spend more time focusing on the product. I feel like I had too much fun with the tool since it is a tool that yu can get distracted with since it has a lot of sounds to play. I think spending more time on the coding would have greatly improved the quality of my product and I will make sure to implement this idea into my next projects.
